# 김치두부찌개

## 기본 정보
- 난이도: 쉬움
- 조리 시간: 20분
- 인분: 2인분

## 재료
| 재료 | 수량 |
|------|------|
| 두부 | 1모 |
| 김치 | 200g |
| 대파 | 1대 |
| 마늘 | 3쪽 (다진 마늘) |
| 돼지고기 목살 | 100g |
| 고춧가루 | 1큰술 |
| 국간장 | 1큰술 |
| 참기름 | 1큰술 |
| 물 | 400ml |

## 조리 순서
1. 김치는 먹기 좋은 크기로 썰고, 돼지고기 목살은 한입 크기로 자른다.
2. 두부는 2cm 두께로 썰고, 대파는 어슷썰기 한다.
3. 냄비에 참기름을 두르고 돼지고기 목살을 중불에서 볶는다.
4. 고기가 반쯤 익으면 김치를 넣고 함께 3분간 볶는다.
5. 고춧가루와 다진 마늘을 넣고 30초간 더 볶아 향을 낸다.
6. 물 400ml를 붓고 센 불에서 끓인다.
7. 끓어오르면 중불로 줄이고 두부를 넣어 10분간 끓인다.
8. 국간장으로 간을 맞추고 대파를 올린 뒤 1분 더 끓여 마무리한다.

## 팁
- 김치는 잘 익은 신김치를 사용하면 국물 맛이 더 깊어진다.
- 돼지고기를 먼저 볶아야 잡내가 줄고 감칠맛이 올라간다.
- 두부는 너무 일찍 넣으면 부서지므로 국물이 끓은 뒤에 넣는다.
- 밥과 함께 먹으면 최고! 계란밥과 곁들이면 더욱 맛있다.
