# 냉장고 재료 기반 레시피 추천 스킬

## 실행 순서

1. `week-4/quest/냉장고 재료 기반 레시피 제작 스킬/ingredients/` 폴더의 모든 JSON 파일을 읽어서 현재 보유 재료 목록을 파악한다.
2. 각 JSON 파일의 구조는 다음과 같다:
   - `name`: 재료명
   - `quantity`: 수량
   - `category`: 보관상태 (냉장/냉동/실온)
   - `exp_date`: 유효기간 (YYYY-MM-DD)
3. 유효기간이 임박한 재료(오늘 기준 7일 이내)를 우선적으로 활용하는 레시피를 추천한다.
4. 보유 재료만으로 만들 수 있는 레시피 3가지를 추천한다.
5. 각 레시피는 다음 정보를 포함한다:
   - 요리명
   - 난이도 (쉬움/보통/어려움)
   - 조리 시간
   - 사용 재료 및 수량
   - 조리 순서 (단계별)
   - 팁
6. 사용자가 레시피를 선택하면, `week-4/quest/냉장고 재료 기반 레시피 제작 스킬/recipes/` 폴더에 해당 레시피를 마크다운 파일로 저장한다.
7. 레시피 파일명은 요리명을 영문으로 변환하여 사용한다 (예: `kimchi-jjigae.md`).

## 레시피 마크다운 파일 형식

```markdown
# {요리명}

## 기본 정보
- 난이도: {난이도}
- 조리 시간: {조리 시간}
- 인분: {인분}

## 재료
| 재료 | 수량 |
|------|------|
| {재료명} | {수량} |

## 조리 순서
1. {단계 1}
2. {단계 2}
...

## 팁
- {팁 내용}
```

## 주의사항
- 반드시 ingredients 폴더의 JSON 파일을 먼저 읽어야 한다.
- 유효기간이 지난 재료는 사용하지 않는다.
- recipes 폴더가 없으면 자동으로 생성한다.
- 사용자에게 추천 레시피 3가지를 보여주고, 선택을 받은 후 레시피 파일을 생성한다.
