## 1. 누구에게 — 핵심 타깃 페르소나

### ① 주말농장 부부 (35~45세, 1순위)
- **상황**: 도시 외곽 주말농장 6~10평 임차, 매주 토~일 한나절
- **고민**: 4월에 모종 시기 놓침, 비료 잘못 줘서 작물 시들음, 카톡 단톡방으로 사진만 흘러감
- **꼬꼬마텃밭에서 얻는 것**: 금요일 저녁 푸시 → "이번 주말 할 일" / 일요일 저녁에 사진 + 한 줄 일지 / 모종·비료 한 번에 결제

### ② 도시 마당·옥상 텃밭 (30~40세, 2순위)
- **상황**: 주택 마당 1~3평 또는 옥상 화분
- **고민**: 인스타에 자랑하고 싶은데 정리가 안 됨
- **꼬꼬마텃밭에서 얻는 것**: 사진 정리 + 친구 공개 일지 + 카카오 공유 한 번에

### ③ 귀촌 시작 1~3년차 (40~55세, 3순위)
- **상황**: 시골 이주 후 100~300평 텃밭
- **고민**: 농사로는 어려움, 비용 추적이 안 됨
- **꼬꼬마텃밭에서 얻는 것**: 가계부로 흑/적자 시각화 + 게시판으로 비슷한 처지 사람들과 소통

### ④ 시골 부모님 + 도시 자녀 (60~75세 / 자녀 30~40대, 보조)
- **상황**: 부모님이 작은 텃밭, 자녀가 도시에서 멀리 살음
- **고민**: 자녀가 부모님 텃밭 진행 상황 보고 싶음
- **꼬꼬마텃밭에서 얻는 것**: 부모님이 사진 1장 올리면 자녀에게 알림 (Care Share)

### 비목표 페르소나 (놓아주기)
- 100평 이상 영농인·전업농 → 팜모닝이 더 잘 맞음
- 농업 R&D 종사자 → 농사로가 더 잘 맞음
- 식물 식별 위주 사용자 → Planta가 더 잘 맞음

## 3. 어떻게 알릴 것인가 — 외부 도달 전략

### A. 카카오톡 친구 공유 (1순위 동선)
- 지인 N명에게 링크 발송이 핵심 가설
- 모든 페이지 우상단에 **카카오 공유 버튼** 노출 (메인·작목 가이드·내 일지·게시판 글)
- 공유 링크에 `share_token` 발급 → 클릭/가입 추적
- 공유 시 카카오 카드 미리보기:
  - 메인: "내가 키우는 작은 텃밭 — 꼬꼬마텃밭"
  - 작목 가이드: "🌱 4월에 심기 좋은 작물 모음"
  - 일지: "🌿 [친구] 의 텃밭 일기 — 4월 셋째 주"
- 발신자는 "초대장" 카드로 발송 (가입 유도 우선)
- **가입 전환 추적 KPI**: 공유 링크 클릭 → 가입까지 12% (6개월 목표) → 18% (12개월)

### B. 비로그인 진입 콘텐츠 (SEO + 자연 유입)
- **공개 페이지** 구조 → 검색 노출 잡기
  - `/` — 이번 달 주말농장 할 일 (월별 자동 갱신)
  - `/calendar/2026-04` — 4월 텃밭 캘린더 (월별 영구 URL)
  - `/crops/{id}` — 각 작목 상세 (12개 → 100개로 단계 확장)
  - `/board/...` — 게시판 글 (사용자 제작 콘텐츠)
- 메타 태그·OG 이미지 최적화 → 카카오/네이버 검색 노출
- **시즌 진입 유도**: 3월에 "4월에 심기 좋은 작물 12선" 콘텐츠 노출

### C. 인스타그램 / 블로그 / 유튜브 (2순위)
- 도시 텃밭러는 **사진을 SNS에 올리는 습관**이 강함
- 일지 페이지에 "이미지 다운로드" 버튼 → 사진 + 한 줄 메모 인스타용 9:16 카드 자동 생성
- 본인 텃밭 일지를 그대로 인스타에 옮기게 → 자연스러운 외부 노출

### D. 네이버 카페·블로그 (3순위, 시간 여유 시)
- "도시농부", "주말농장" 카페 일주일 1회 양질 콘텐츠 글 (광고 X, 가이드 위주)
- 블로그 운영자 5~10명에게 무료 베타 액세스 제공 → 후기 유도

### E. 오프라인 (시즌 한정)
- 4월 초 모종시장에서 QR 스티커 배포 (지역 5곳)
- 주말농장 임대 사업자 제휴 (계약자 50명에게 가입 권유)

## 4. 시기 알림 체계 — 사용자가 떠나지 않게 하는 핵심

### 채널 우선순위
1. **로그인 팝업** (모든 회원, 즉시 노출) — 1순위
2. **이메일 다이제스트** (선택 회원, 주 1회 금요일 저녁) — 2순위
3. **Web Push** (선택 회원, v1.5에서 도입) — 3순위
4. **카카오톡 알림톡** (예산 허락 시 v2) — 4순위

### 트리거 룰
- **시즌 트리거**: 작목별 모종 시기 (예: 고추 = 4월 셋째 주~다섯째 주)
- **사용자 트리거**: 내 작물 등록 후 N주차 (예: 모종 심은 후 2주차 = 추비 시기)
- **날씨 트리거**: 시·도별 일주일 비 예보 → "이번 주 비 옴, 비료 한 박자 늦추세요"
- **빈 일지 트리거**: 마지막 일지 후 14일 → "이번 주말 어떠셨나요?" (라이트하게)

### 노이즈 방지
- **주 1회 캡** — 한 사용자에게 알림 ≤ 1/주
- **다시 보지 않기** 토글
- **알림 받기 종료** 1클릭

### 마이크로카피 톤 (꼬꼬마텃밭 답게)
- 좋음: "이번 주말은 고추 모종 심기 좋은 때예요 🌱" *(이모지는 알림 안만)*
- 좋음: "4월 셋째 주 — 상추는 이제 수확해도 좋아요"
- 피함: "시급한 농작업 알림" / "[중요] 즉시 확인" 같은 푸시-푸시한 톤

## 5. 커뮤니티 운영 — 가입자간 소통

### 게시판 4개 보드
| 보드 | 용도 | 예시 |
|---|---|---|
| **질문** | 답변 받기 | "잎이 누레요, 비료 부족인가요?" |
| **자랑** | 사진 공유 | "오늘 첫 수확 🌶️" |
| **정보** | 팁 공유 | "4월 모종 살 때 체크리스트" |
| **자유** | 잡담 | "이번 주말 비 와서 못 갔네요" |

### 댓글 + 좋아요 + 신고
- 댓글: 1차 댓글 + 1단계 대댓글 (깊이 제한)
- 좋아요: 게시글·댓글
- 신고: 욕설/광고/스팸 → 관리자 큐 → 24시간 내 처리
- 가입 후 **3일 이내** 게시글 작성 제한 (스팸 방지)

### 농사일기 공유 (커뮤니티 핵심 동선)
- 일지 작성 시 "전체 공개"로 설정 → "자랑" 보드에 동시 노출
- 다른 사용자가 일지에 댓글 가능
- "👍 비슷한 텃밭 보기" 추천 (지역·작목 매칭)

### 활동 인증 / 게이미피케이션 (v1에 가볍게)
- "이번주 농작업 + N명" 카운터 (팜모닝 패턴 차용)
- 첫 일지·첫 결제·첫 댓글 작성 시 작은 배지

## 6. 비로그인 → 로그인 → 결제 깔때기

```
비로그인 방문 (월 5,000 목표)
   │  카카오 공유 / 검색 / SNS
   ▼
캘린더·작목 가이드 열람 (이탈률 50%)
   │  "내 작물 등록하기" CTA
   ▼
회원가입 (전환 25%, 1,250명)
   │  로그인 팝업 알림
   ▼

   │  주 1회 알림 → 재방문
   ▼

   │
   ▼
자재 결제 1회 (회원의 5%, 63명/월) ← 매출
   │
   ▼
다음 시즌 복귀 (12개월 35% 목표)
```

## 7. 6개월 / 12개월 성장 로드맵

### 1~2개월 (출시 직후)
- 지인 50명에게 카카오 직접 공유
- 인스타 텃밭 해시태그 글 10건
- 게시판에 "운영자 손글씨" 콘텐츠 주 2회 (커뮤니티 활성화 시드)
- **목표**: 가입 200, WAU 50

### 3~4개월 (시즌 정점, 5~6월)
- 4월 모종시장 QR 배포
- "4월 텃밭 캘린더" 검색 SEO 잡기
- 베타 후기 10건 → 블로그 / 인스타
- **목표**: 가입 500, WAU 100

### 5~6개월 (수확 시즌)
- "수확 자랑" 이벤트 — 인스타 해시태그 #꼬꼬마텃밭 사용 시 자재 쿠폰
- 부모님-자녀 Care Share 사용 사례 마케팅
- **목표**: 가입 800, WAU 150, 결제 25/월

### 7~9개월 (비시즌, 11~1월)
- 사용자 이탈 심함 → 콘텐츠로 유지
- "내년 농사 계획" 가이드 + 다음 시즌 사전 알림
- 종자 사전 예약 결제 (소액)

### 10~12개월 (시즌 복귀, 2~4월)
- 작년 가입자에게 "올해는 시작 안 하시나요?" 알림 (조심스러운 톤)
- 신규 작목 추가 (12 → 30개)
- **목표**: 가입 누적 4,000, WAU 1,000, 시즌 복귀율 35%

## 10. 한 페이지 결론

> **꼬꼬마텃밭은 카카오 친구 공유 한 번에서 시작한다.**
> 비로그인이라도 1년 캘린더와 작목 가이드를 열람하게 풀어두고, 가입하면 일지·가계부·결제·알림으로 매주 돌아오게 만든다. 게시판은 도시 주말농장러끼리 따뜻한 톤으로 묶이는 광장이 되고, 일지의 "친구 공개" + 카카오 공유 동선이 자연스럽게 새 사용자를 데려온다. **시즌 복귀율 35%**가 12개월 후의 진짜 시험대.
