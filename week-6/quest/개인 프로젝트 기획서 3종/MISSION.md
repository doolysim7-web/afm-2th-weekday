# MISSION.md — 꼬꼬마텃밭 (LittleFarm)

> 작성일: 2026-04-26
> **앱 이름**: 꼬꼬마텃밭 *(영문: LittleFarm)*
> **한 줄 미션**: "토요일 아침에 텃밭 가서 뭐 해야 하는지, 일요일 저녁에 어떻게 정리해두는지 — 이걸 한 손바닥 안에서."
> **컨셉**: 주말농장 · 작은 텃밭 · 도시 농부의 따뜻한 동반자

## 0. 앱 이름 — 왜 "꼬꼬마텃밭"?

| 후보 | 의미 / 느낌 | 선정 |
|---|---|---|
| **꼬꼬마텃밭** | 꼬꼬마(아주 작고 귀여운) + 텃밭(소규모 밭) | ⭐ **최종 선정** |
| 한뼘텃밭 | 손바닥 한 뼘 만한 작은 텃밭 | 후보 |
| 모종이네 | 모종(seedling) + ~이네(친근한 가족 호칭) | 후보 |
| 새싹이네 | 새싹(sprout) + ~이네 | 후보 |
| 텃밭콩콩 | 콩콩 뛰어가는 텃밭 | 후보 |

**꼬꼬마텃밭**으로 정한 이유:
- "꼬꼬마"는 **작다·귀엽다·아끼다**라는 정서를 가장 잘 담는 한국어 (어린이를 부르는 따뜻한 말)
- "주말 텃밭, 한 평짜리 작은 농사"라는 컨셉을 그대로 옮김
- 영문 브랜드 **LittleFarm**으로 직역 가능 → DB prefix `little_farm_`과 일치
- 검색 노출 시 차별화: "텃밭 + 일기" 검색 결과에서 톤이 다름 (기존 앱들은 무뚝뚝)

**브랜드 톤**: 손편지 같은 따뜻한 한국어, 큰 글씨, 둥근 모서리, 파스텔(연두/베이지/주황). "어르신 서비스" 느낌은 피하면서도 **친절하고 다정한 말투**를 유지.

## 1. 왜 만드는가 — 진짜 문제

주말농장·작은 텃밭을 처음 시작한 사람은 매번 같은 벽에 부딪힌다.

1. **"이번 주말에 가서 뭘 해야 하지?"**
   금요일 밤에 검색해보면 농진청 자료는 어렵고, 블로그는 흩어져 있고, 동영상은 길다. 결국 토요일 아침 텃밭 가서 멍하니 서 있게 됨.

2. **"내 텃밭에 뭘, 언제, 얼마나 심어야 하지?"**
   주말농장은 보통 5~15평. 모종 몇 개 사야 하는지, 줄 간격은 얼마인지, 비료는 얼마나 줘야 하는지 — 정보는 다 평·헥타르 단위라 환산이 어렵다.

3. **"저번 주에 뭘 했더라?"**
   심은 날, 비료 준 날, 풀 뽑은 날 — 핸드폰 사진첩과 카톡 메모에 흩어져 있음. 다음 해엔 또 처음부터.

4. **"올해 텃밭에 얼마 썼지? 사 먹는 게 더 싸지 않아?"**
   임차료, 모종, 비료, 호미, 장갑… 영수증 정리 안 함. 1년 끝나고 보면 "재밌었지만 얼마 들었는지 모름".

5. **"같이 시작한 친구들과 어떻게 정보 나누지?"**
   카톡 단톡방으로는 흩어지고, 농민 카페는 분위기가 안 맞는다(전업농 위주).

**결과**: 매년 같은 시행착오 → 모종 시기 놓침, 비료 잘못 줘서 시들음, 주말 농장이 그냥 흐릿한 추억으로 남음.

## 2. 기존 솔루션의 빈틈 (Chrome MCP 직접 분석, 2026-04-26)

### A. 농사로 (nongsaro.go.kr) — 농촌진흥청 공식 포털
- **강점**: 작목별 깊이 있는 가이드, AI 이삭이 챗봇, 병해충 AI 영상 진단, 토양 정보(흙토람), 농업기상정보. 정부가 가진 모든 데이터.
- **빈틈** (= 우리가 노릴 곳):
  - 학술 논문 같은 톤 → 주말농장러에게는 너무 무거움
  - **개인 농사일지 기능 없음**, 비용 관리 없음
  - 모바일 메뉴 깊고 PC 중심
  - 평·헥타르 단위 정보 → 5평짜리 텃밭에 적용하기 어려움
  - 친구 공유 동선 거의 없음

### B. 팜모닝 (farmmorning.com) — 농민 커뮤니티 앱
- **강점**: 활성 사용자 많음(이번주 농작업 3,090명), 보조금 계산기, 장터, 병해충 사진 즉답.
- **빈틈** (= 우리가 노릴 곳):
  - **고연령 전업농 위주 톤** — 30~50대 도시인이 진입하기 어색함
  - 1년 농사 시기를 한눈에 보여주는 캘린더 없음 (작목별 분산)
  - **농사 가계부 없음** — 영농일지는 있지만 돈 흐름은 추적 X
  - 외부 카카오 공유 동선 약함
  - 비로그인 방문자 진입 콘텐츠 약함

### C. Planta (getplanta.com) — 글로벌 식물 케어 앱 (7M+ 사용자)
- **강점**: 30+ 파라미터 + 지역 기상 기반 **개인화 알림**(물·비료·분무·분갈이), 사진 진단(Dr Planta), 식물 식별, **Care Share(가족과 일정 공유)**, Light Meter.
- **빈틈** (= 우리가 노릴 곳):
  - 반려식물(실내) 중심 → 한국 노지 텃밭 작물 데이터 X
  - 한국어 미지원, 절기·기후·비료 한국 기준 부재
  - 한국 결제 없음, 한국 자재 구매 연계 없음

## 4. 타깃 사용자 페르소나

### 페르소나 ① 주말농장 부부 (35~45세, 1순위)
- 도시 외곽 주말농장 6~10평 임차 (월 2~5만원)
- 매주 토~일 한나절 들름. 비료·모종은 인근 종묘상에서 충동 구매
- 매년 같은 실수: 4월에 모종을 못 사서 5월에야 시작 → 한 박자 늦음
- 카톡 단톡방 친구들과 농작물 사진 공유
- **결제 의사**: 모종·자재 구색이 갖춰져 있으면 한 번에 사고 싶어함

### 페르소나 ② 도시 마당·옥상 텃밭 (30~40세, 2순위)
- 주택·빌라 마당 1~3평, 또는 옥상 화분 텃밭
- 인스타그램에 사진 올리는 게 즐거움
- "내가 키운 거 먹는 만족"이 핵심 동기
- 친구들에게 "이거 나도 하고 싶어" 자랑하고 싶음

### 페르소나 ③ 귀촌 시작 1~3년차 (40~55세, 3순위)
- 시골 이주 후 100~300평 텃밭
- 농사로는 어려워서 포기, 동네 어르신·유튜브로 학습
- 비용 추적이 안 됨 → "흑자인지 적자인지 모름"
- 우리 앱이 텃밭 규모용이지만, 가계부·일지·결제는 잘 맞음

### 페르소나 ④ 부모님께 선물 (자녀가 깔아드림, 보조)
- 시골 부모님이 작은 텃밭 가꿈
- 자녀가 도시에서 진행 상황 보고 싶음 → **Care Share** 시나리오
- 알림은 자녀에게도 동시 발송

> ⚠️ **비목표 페르소나**: 100평 이상 영농인·전업농. 팜모닝이 더 잘 맞음.

## 6. MVP에 포함하지 않을 것 (의도적 비목표)

- 작물 자동 식별 카메라 (Planta 수준의 ML 모델은 비용 큼)
- 병해충 AI 영상 진단 (농진청 API로 v2에서 연계)
- 직불금/보조금 계산기 (팜모닝 영역, 우리 사용자에겐 불필요)
- 실시간 경매 시세
- 대규모 영농 ERP (회계·세무)
- 안드로이드/iOS 네이티브 (v1은 모바일 웹 + PWA)

## 한 페이지 결론

> **꼬꼬마텃밭** — 주말농장러를 위한 따뜻한 농사 동반자.
> 농사로의 정확함 + 팜모닝의 커뮤니티 + Planta의 개인화를, **5~10평 텃밭 단위로 환산하고 친절한 톤으로** 다시 짠다. 토요일 아침에 펼치면 이번 주말에 뭘 해야 하는지 알려주고, 일요일 저녁에 사진과 비용을 한 줄로 남긴다. 그게 미션.
